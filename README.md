Overview

Usage
---------

1. Install
`$ cd ~`
`$ git clone xxx/dots.git .`
3. Set Environment
`$ .bin/dotsinstall.sh`
4. vim plugin install
`$ vi`
`press y`
5. exec zsh
6. Enjoy!
